- Tailwind

## Cara Install dan Menjalankan Proyek
1. git clone https://github.com/arifwp/rimba-fe
2. cd <NAMA_FOLDER_PROJECT>
3. npm install
4. Rename file .env.example menjadi .env
5. npm start
